# Bundled skills (reference)

This folder includes a **vendored copy** of Anthropic’s public skills repository:

- **Path:** `anthropic-official/`
- **Upstream:** [github.com/anthropics/skills](https://github.com/anthropics/skills)

## License & usage

Skills in that tree are a mix of **Apache-2.0** examples and **source-available** document skills (`docx`, `pdf`, `pptx`, `xlsx`). See each skill’s `LICENSE.txt` where present, and the upstream `README.md`.

**You do not have to use these files.** They are here so you can:

- Copy patterns into your own `.cursor/skills/` or project docs  
- Learn how `SKILL.md` frontmatter and instructions are structured  
- Remove the entire `anthropic-official/` directory if you want a minimal repo  

For the Agent Skills standard, see [agentskills.io](https://agentskills.io).
